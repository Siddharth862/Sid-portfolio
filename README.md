# Siddharth Gupta - Personal Portfolio

A modern, fully responsive personal portfolio website showcasing my skills, enterprise experience, and projects as a Full-Stack Developer. Built with a clean, dark-themed UI and smooth scroll animations.

## 🚀 Features

*   **Modern Dark UI:** Sleek, professional design utilizing CSS variables for consistent theming and a subtle noise overlay.
*   **Fully Responsive:** Fluid layout optimized for desktops, tablets, and mobile devices using CSS Grid and Flexbox.
*   **Smooth Animations:** Performance-friendly "fade-up" scroll animations and dynamic skill bar fills powered by the Intersection Observer API.
*   **Interactive Elements:** Mobile hamburger menu, sticky navigation bar with background blur, and a simulated contact form with visual feedback.
*   **Organized Codebase:** Clean separation of concerns across HTML, CSS, and JavaScript files for easy maintenance.

## 🛠️ Tech Stack

*   **HTML5:** Semantic markup and structure.
*   **CSS3:** Custom properties (variables), transitions, responsive media queries, and Font Awesome icons.
*   **Vanilla JavaScript:** DOM manipulation, event listeners, and Intersection Observers for scroll-triggered animations.

💻 Getting Started
To view or edit this project locally:

Clone or download the repository to your local machine.

Ensure index.html, style.css, and script.js are all located in the same root directory.

Open index.html in any modern web browser (Chrome, Firefox, Safari, Edge). No local server or build tools are required.

🎨 Customization Guide
Theme Colors: You can easily change the primary color scheme by modifying the --accent, --accent2, and --accent3 variables located in the :root block at the very top of style.css.

Content: Update the text, skills, and project descriptions directly inside index.html.

Resume Download: Create an assets/pdf/ folder in your root directory and add your resume document to ensure the "Download resume" button works correctly.
